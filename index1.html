Question-1: What are the different ways to select an element in the DOM?

Answer: There are many ways to select an element in the DOM, such as
"getElementsByTagName", "getElementsByClssName", "getElementById", "querySelector" for single selection like id selector, "querySelectorAll" for all like class selector etc.

Question-4: What is event bubbling in the DOM?

Answer: event bubbling বলতে বুঝি, child element-এ ঘটা ইভেন্ট parent elements-এর দিকে ছড়িয়ে পরা কে। অর্থাৎ, চাইল্ডে একটা ক্লিক বা যেকোনো ইভেন্ট ঘটাইলাম প্রথমে সেটা তার প্যারেন্ট দিকে যাবে তারপর তার প্যারেন্টে তারপর তার প্যারেন্টে এভাবে একদম HTML এবং সবশেষে document এ। আমি জানি ভালভাবে বুঝাতে পারিনি, আমি যত দ্রুত সম্ভব আমার ল্যাকিংস গুলো পুরন করে ফেলব।

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:
To create any element - "document.createElement()"
To add any element - ".appendChild()" or ".append()"
To remove any element - ".removeChild()" or ".remove()"
